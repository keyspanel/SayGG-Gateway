This is an existing React + TypeScript + Express + PostgreSQL payment gateway project. It currently works as a single-user merchant gateway. I want to upgrade it into a professional role-based, owner-based, paid-plan gateway SaaS system.

Do NOT rebuild from scratch.
Do NOT damage the current working payment gateway.
Do NOT remove existing payment order, hosted page, QR, redirect_url, cancel_url, callback_url, token, UPI setup, or transaction features.
Refactor safely and migrate existing data without breaking production.

The final system must support:

1. Owner role
2. Normal user role
3. Paid subscription plans
4. Separate Method 1 and Method 2
5. Owner-managed plan prices, discounts, durations
6. Owner UPI setup for plan purchase payments
7. User plan purchase flow using hosted billing pay page
8. Backend-enforced feature restrictions
9. Professional owner panel
10. Professional user billing/plan pages
11. Clean API docs based on the user's active plan

==================================================
MAIN CONCEPT
==================================================

Currently the project has one merged payment method.

Now split it into 3 paid plans:

Plan 1: Method 1 — Server API
- User can use server-to-server payment API only.
- User can create orders with mode="server".
- User gets raw UPI payload / payment link for own frontend rendering.
- User can use callback_url webhook.
- User can use check-order.
- User cannot use hosted pay page.
- User cannot get payment_page_url.
- User cannot use redirect_url/cancel_url hosted page flow.

Plan 2: Method 2 — Hosted Pay Page
- User can create hosted payment links only.
- User can create orders with mode="hosted".
- User gets payment_page_url.
- User can use redirect_url and cancel_url.
- User can use hosted pay page.
- User cannot get raw UPI payload for server-only rendering.

Plan 3: Master Plan
- User can use both Method 1 and Method 2.
- User can create orders with mode="server" or mode="hosted".

Owner:
- Owner can use everything without plan restrictions.
- Owner can manage users, plans, prices, discounts, subscriptions, and platform UPI settings.
